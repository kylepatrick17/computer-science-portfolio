The motherboard contains a lot of important systems under and on top of it. The cpu controls the core elements. cpu stands for central processing unit. The next component is the ram or random acess memory where the applications and files currently being used are kept.Next there is the hard drive where all the information is stored. All the compenents interact through the electric circut. The keyboard is used for typing. The mouse is used for scrolling on the screen. The monitor is used to display the images.
